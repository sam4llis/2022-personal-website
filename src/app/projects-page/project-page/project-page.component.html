A novel attack was presented in this research, which used cross-prediction techniques on a stereo array of microphones to increase keystroke recovery rate by 15% when compared to previous research papers. It was shown from my research that key classification using a cross-predicted Random Forest model can achieve up to 89% keystroke recovery rate from a 40-key classification problem.
